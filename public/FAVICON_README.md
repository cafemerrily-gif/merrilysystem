# ファビコンについて

このプロジェクトでは以下のファビコンを使用しています：

## 現在のファイル
- `favicon.svg` - SVG形式のファビコン（モダンブラウザ対応）
- `favicon.ico` - ICO形式のファビコン（従来のブラウザ対応）

## ファビコンの変更方法

### 方法1: オンラインツールを使用
1. https://favicon.io/ にアクセス
2. 好きな画像をアップロード
3. 生成されたファイルをダウンロード
4. `public/` フォルダに配置

### 方法2: 自分で作成
1. 正方形の画像を用意（推奨: 512x512px以上）
2. https://realfavicongenerator.net/ でICO形式に変換
3. 生成されたファイルを `public/` フォルダに配置

## 使用している緑色
- メインカラー: #22c55e (Tailwind CSS の green-500)
- ダークカラー: #16a34a (Tailwind CSS の green-600)
